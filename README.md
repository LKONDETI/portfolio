# Lakshmi Sravani Kondeti – Portfolio

A modern, responsive portfolio built with **React** and **Tailwind CSS**.  
Showcasing experience in full-stack development, machine learning, and cloud technologies.

---

## ✨ Features

- **Animated Hero Section** with gradient backgrounds and personal intro
- **Navigation** for About, Experience, Projects, Skills, and Education
- **Experience Timeline** with highlights and roles
- **Project Gallery** with tech stack and key achievements
- **Skills Grid** for languages, frameworks, databases, tools, and ML
- **Education & Certifications** section
- **Responsive Design** for all devices
- **Modern UI** with Tailwind CSS, gradients, and subtle animations

---

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/<your-username>/<your-repo>.git
   cd myportfolio
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.
